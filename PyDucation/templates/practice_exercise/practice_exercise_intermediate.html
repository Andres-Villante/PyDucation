Ejercicios Intermedios

En esta sección encontrarás ejercicios de nivel intermedio para poner en práctica tus habilidades en Python:

  1. Ejercicio 1:

    Escribir un programa que reciba una lista de números enteros y devuelva la suma de los elementos de la lista.

    
                    numeros = [1, 2, 3, 4, 5]
                    suma = 0
                    for num in numeros:
                        suma += num
                    print("La suma de los números es:", suma)
  2. Ejercicio 2:

    Escribir una función que reciba un número entero positivo y devuelva su factorial.

    
                    def factorial(num):
                        if num == 0:
                            return 1
                        else:
                            return num * factorial(num - 1)

                    numero = int(input("Introduce un número entero positivo: "))
                    print("El factorial de", numero, "es", factorial(numero))
  3. Ejercicio 3:

    Escribir un programa que lea una cadena de texto y la imprima al revés.

    
                    cadena = input("Introduce una cadena de texto: ")
                    cadena_invertida = cadena[::-1]
                    print("La cadena invertida es:", cadena_invertida)
