Ejercicios Difíciles

En esta sección encontrarás ejercicios de nivel difícil para poner a prueba tus habilidades en Python:

  1. Ejercicio 1:

    Escribir una función que reciba una lista de números como argumento y devuelva una nueva lista con los números ordenados de mayor a menor.

    
                    def ordenar_numeros(lista):
                        lista.sort(reverse=True)
                        return lista
                        
                    numeros = [5, 3, 8, 1, 9, 2]
                    numeros_ordenados = ordenar_numeros(numeros)
                    print(numeros_ordenados)
  2. Ejercicio 2:

    Escribir una función que reciba una lista de palabras como argumento y devuelva una nueva lista con las palabras que empiezan y terminan con la misma letra.

    
                    def palabras_iguales(lista):
                        palabras_iguales = []
                        for palabra in lista:
                            if palabra[0] == palabra[-1]:
                                palabras_iguales.append(palabra)
                        return palabras_iguales
                        
                    palabras = ["reconocer", "hola", "oso", "python"]
                    palabras_iguales = palabras_iguales(palabras)
                    print(palabras_iguales)
  3. Ejercicio 3:

    Escribir una función que reciba una cadena de texto como argumento y devuelva una nueva cadena de texto con las palabras que contengan todas las vocales.

    
                    def palabras_con_vocales(cadena):
                        palabras_con_vocales = []
                        vocales = ['a', 'e', 'i', 'o', 'u']
                        for palabra in cadena.split():
                            if all(vocal in palabra for vocal in vocales):
                                palabras_con_vocales.append(palabra)
                        return ' '.join(palabras_con_vocales)
                        
                    texto = "La tarea de hoy es encontrar una palabra que tenga todas las vocales."
                    palabras = palabras_con_vocales(texto)
                    print(palabras)
